# Password Security and Best Practices

## 1. What makes a strong password?

A strong password is long, unique, and difficult to guess. It should contain a combination of uppercase letters, lowercase letters, numbers, and special characters. Strong passwords help protect accounts from unauthorized access.

## 2. Why should passwords be unique?

Passwords should be unique because if one account is compromised, attackers cannot use the same password to access other accounts. Using different passwords for each account improves overall security.

## 3. What is Multi-Factor Authentication (MFA)?

Multi-Factor Authentication (MFA) is a security method that requires users to provide two or more forms of verification before accessing an account. For example, after entering a password, a user may need to enter a code sent to their phone.

## 4. What are the risks of password reuse?

Password reuse increases security risks. If a hacker obtains a password from one website, they may try the same password on other accounts such as email, banking, or social media. This can lead to multiple accounts being compromised.

## Password Policy – Best Practices

1. Use at least 12 characters in every password.
2. Use a combination of uppercase and lowercase letters.
3. Include numbers and special characters.
4. Avoid using personal information such as names, birthdays, or phone numbers.
5. Use a unique password for each account.
6. Enable Multi-Factor Authentication (MFA) whenever possible.
7. Do not share passwords with others.
8. Change passwords immediately if a compromise is suspected.
9. Use a trusted password manager to store passwords securely.
10. Avoid writing passwords in easily accessible locations.

## Conclusion

Following strong password practices and enabling MFA significantly improves account security. Unique and complex passwords help protect personal and professional information from cyber threats.
