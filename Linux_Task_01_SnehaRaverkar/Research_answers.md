# 1. What is Linux?
Linux is an operating system that helps a computer work properly. It manages hardware,
software, files, and programs. Linux is free to use and is popular because it is secure and
reliable.

# 2. Why is Linux important in Cyber Security?
Linux is important in Cyber Security because many security tools run on Linux. It allows users to
perform security testing, network analysis, and system monitoring more effectively. Many
servers also use Linux, so learning Linux is important for cybersecurity professionals.

# 3. Difference between Linux and Windows
Linux is a free and open-source operating system, while Windows is developed by Microsoft
and requires a license. Linux provides more customization and is widely used by programmers
and cybersecurity experts. Windows is generally easier for beginners and is commonly used on
personal computers.

# 4. What is a Linux Distribution?
A Linux Distribution is a version of Linux that includes the operating system along with useful
software and tools. Different distributions are designed for different purposes. Examples include
Kali Linux, Ubuntu, and Parrot OS.

# 5. Why do Ethical Hackers prefer Linux-based operating systems?
Ethical hackers prefer Linux because it provides powerful security tools, better control over the
system, and strong command-line support. Many penetration-testing and cybersecurity tools are
available on Linux, making it a preferred choice for ethical hacking and security research.
